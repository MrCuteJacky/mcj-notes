## Notes
「[Notes](https://mrcutejacky.github.io/mcj-notes/)」是使用Angular開發的服務，透過Google Sheets API存資料。

### Tools
1. Angular 6
2. Google Sheets API
3. GitHub Pages

### Development environment
```console
npm install
```

### Run in local
```console
ng serve
```

### Build Production
```console
ng build --prod --output-path docs --base-href
```
